# PostgreSQL Procedures Migration - Audit Spreadsheet

Full inventory of Python-PostgreSQL touchpoints with proposed procedure/function mapping.
Generated for the PostgreSQL Procedures and Functions Migration Plan.

## Scope

Every `session.execute()`, `select()`, `update()`, `session.merge()`, `session.delete()`, and
`text()` call that touches PostgreSQL in production code (excluding tests).
